# Kinda sorting Trello

## Requirements

- Node.js v8.7.0 +
- TypeScript v3.0.0 +

## Installation

- **Install NodeJs from** [https://nodejs.org/uk/](https://nodejs.org/uk/)
- **npm i -g typescript**
- **npm i**

## Run Tests

- **npm test**

## Run Application

- **npm start**

## Build Application For Production

- **npm run build**
